声明
声明我要的是什么。
命令
我要这个我要干什么
```js
var arrays = [1,2,3,4]
want to  be [2,4,6,8]
let len = arrays.length
for (let i =0;i<len;i++){
    arrays[i] *=2
}
```
命令式。我们指定指出每一步干什么，怎么干，强调了过程

arrays.map((value)=>{return value*2})
声明式，我们指定了目标。我们要什么。强调了结果，目的


函数式变成实际上就是声明式编程的子编程范式
摘自wiki
函数式编程，特别是纯函数式编程，尝试最小化状态带来的副作用，因此被认为是宣告式的(声明式)
