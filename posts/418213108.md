### 实实在在解决了问题

1. 组件多层wrapper
2. 生命周期方法的共用

<xxx.consumer>
{
  props => xxxx
}
</xxx.consumer>

---
const xxx = useContext(xxx)
组件只要使用这个xxx就可以了
少了嵌套使用 

对函数订阅等方法，都可以使用hook进行公用，从此函数就是函数。而不用作为类的方法。

这与mixin相似，但是却不是mixin实现、



不能在一个条件判断中使用

if (condition) {
  const xxx = useContext(xxx)
}
