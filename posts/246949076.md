在React使用JSX编写dom
约定大小写来区分html原生标签与React组件
```
import React,{Component} from 'react'

class TestTemp extends Component{
    constructor () {
        super()
    }
    render (){
        return (
            <h1>React Test Template</h1>
        )
    }
}
```
这样子的就是一个React很简单的一个组件demo
